StringModifier

enum StringModifier : Enum<StringModifier>

Modifier for string comparison operators that controls case and diacritic sensitivity.

By default, NSPredicate string comparisons are case and diacritic sensitive. Use these modifiers to change that behavior.

Example:

name.beginsWith("bar", CaseAndDiacriticInsensitive)
// Produces: name BEGINSWITH[cd] 'bar'
