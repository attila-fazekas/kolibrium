XCUIElementType

object XCUIElementType

Constants for XCUITest element type names used in iOS locator strategies.

These correspond to the cases of Apple's XCUIElement.ElementType enumeration. Use them with dev.kolibrium.appium.dsl.ios.nsPredicate or iOSClassChain to avoid error-prone raw strings.

Example:

private val login by iOSNSPredicate(nsPredicate {
    type equalTo XCUIElementType.BUTTON
    label equalTo "Login"
})
