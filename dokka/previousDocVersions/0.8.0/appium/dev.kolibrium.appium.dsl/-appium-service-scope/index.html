AppiumServiceScope

class AppiumServiceScope

DSL scope for configuring and building an AppiumDriverLocalService.

This scope mirrors the common configuration options available on AppiumServiceBuilder and exposes a concise, type-safe configuration style. Use it via the top-level appiumService function.

Example:

val service = appiumService {
    ipAddress = "127.0.0.1"
    port = 4723
    logLevel = "info"
    environments { environment("ANDROID_HOME", "/path/to/sdk") }
}
