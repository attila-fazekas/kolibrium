Page

abstract class Page<S : Site> : SearchContext

Base type for page objects bound to a Site.

Page instances rely on a contextual WebDriver installed by Kolibrium's DSL. Constructing and using pages outside those contexts will fail with a runtime error. Delegated findElement(s) calls route through the contextual driver.

path

open val path: String

Relative path of this page within the Site. Defaults to empty string (no navigation). Any non-empty value causes the dev.kolibrium.selenium.dsl.SiteScope.on function to navigate to baseUrl + path.
