AbstractElementDescriptor

abstract class AbstractElementDescriptor<T : AbstractElementDescriptor<T, R>, R>(searchCtx: SearchContext)

Base implementation for element locator delegates with built‑in waiting.

This abstract class powers Kolibrium's locator delegates by:

  • Providing a consistent wait loop via Selenium's FluentWait, configured by WaitConfig

  • Ignoring transient lookup errors: always ignores NoSuchElementException and handles StaleElementReferenceException by clearing cache and retrying

Subclasses implement how to locate the element(s) (findElement), how to drop any cache (clearCache), and how to determine element readiness (isElementReady).

Parameters

searchCtx

The original Selenium SearchContext used as the base for all lookups.

Type Parameters

T

The self type of the concrete descriptor (used to type the FluentWait).

R

The result element type returned by this descriptor (e.g., org.openqa.selenium.WebElement or dev.kolibrium.webdriver.WebElements).
