SingleElementDescriptor

open class SingleElementDescriptor(searchCtx: SearchContext, value: String, locatorStrategy: (String) -> By, cacheLookup: Boolean, waitConfig: WaitConfig, readyWhen: WebElement.() -> Boolean) : AbstractElementDescriptor<SingleElementDescriptor, WebElement> , WebElementDescriptor

Descriptor/delegate for lazily locating a single WebElement.

Supports optional caching of the located element, configurable waiting via waitConfig, and a per‑element readiness predicate via readyWhen. The element is searched using the provided locatorStrategy against the SearchContext.

Typical usage is via higher‑level helpers (e.g., id, cssSelector, and Appium's accessibilityId) which supply value and locatorStrategy.

Parameters

searchCtx

The base SearchContext to perform the lookup in (driver/page/screen or a nested element).

value

The raw locator value (e.g., CSS, XPath, id) passed to locatorStrategy. Must be non‑blank.

locatorStrategy

Function that converts value into a Selenium By.

cacheLookup

When true, cache the first resolved WebElement and reuse it until cache is cleared.

waitConfig

WaitConfig to control timeout/polling/ignored exceptions.

readyWhen

Predicate that defines when the found element is considered ready.
